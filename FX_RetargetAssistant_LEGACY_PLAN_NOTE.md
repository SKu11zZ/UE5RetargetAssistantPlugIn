# FX_RetargetAssistant Legacy Plan Archive

This document archives earlier planning ideas.

Current active direction:

- Unreal Engine 5.8 only
- MVP1 Alpha UE5.8 Migration
- Guided Auto Setup + Batch Export

The older cross-version plan is frozen. The project is no longer targeting UE5.4 - UE5.8 compatibility in the current mainline.

Archived ideas:

- Multi-version support for UE5.4 / 5.5 / 5.6 / 5.7 / 5.8.
- Expanding `VersionBridge` for ongoing compatibility.
- Maintaining per-version BuildPlugin packages.
- Treating MVP0 "existing Retargeter only" as the current product flow.

Current rule:

MVP0 is historical validation. MVP1 is the active workflow, while Manual Mode remains available as a read-only user Retargeter path.
